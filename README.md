# PulseConsole

PulseConsole is a script-driven, console-like text renderer for stage presentations.

## Tech
- C++23
- Qt 6 (Widgets)
- YAML (yaml-cpp)

## Build (Windows / MSVC)
1. Install Qt 6.x (MSVC 2022 kit).
2. Configure CMake with:
   -DCMAKE_PREFIX_PATH=C:/Qt/6.7.3/msvc2022_64
3. Build and run from CLion.
